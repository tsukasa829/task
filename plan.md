# Task Manager 開発計画

## 実装予定機能

### [ ] フォーム内容のローカルストレージ保存
- フォームの入力内容をlocalStorageに保存
- 次回アクセス時に保存内容を復元
- タスク追加時にフォームをリセット

### [ ] ナレッジテーブルの作成
- シンプルな構造:
  - `id`: 一意識別子
  - `content`: テキスト（内容）
  - `tag`: テキスト（単一タグ、デフォルト: "未分類"）
  - `createdAt`: 作成日時

### [ ] クイック追加にタグボタンを追加
- 直近で追加したタグ名のボタンを表示
- ボタン押下で、そのタグ名でナレッジを即座に保存
- TODOカードにも同様のタグボタンを設置

### [ ] ナレッジの表示
- 期間フィルター（1週間など）に「ナレッジ」項目を追加
- 「ナレッジ」選択時、全てのナレッジを表示
- ナレッジアイテムはTODOリストには表示しない

### [ ] TODOからナレッジへの変換機能
- TODOカードに「ナレッジ」ボタンを追加
- ボタン押下でTODOのtitleをcontentとしてナレッジに移動
- tagは指定されなければ"未分類"をデフォルト設定

# Task Manager 開発計画

## 実装予定機能

### [ ] フォーム内容のローカルストレージ保存
- フォームの入力内容をlocalStorageに保存
- 次回アクセス時に保存内容を復元
- タスク追加時にフォームをリセット

### [ ] ナレッジテーブルの作成
- TODOと同じ項目構造を持つテーブル
- 追加項目:
  - `content`: テキスト（詳細内容）
  - `tag`: テキスト（タグ情報）

### [ ] クイック追加にナレッジ項目を追加
- クイック追加フォームに「ナレッジ」選択肢を追加
- ナレッジとTODOを区別して追加可能に

### [ ] ナレッジの表示分離
- ナレッジアイテムはTODOリストに表示しない
- ナレッジ専用の表示領域を作成

### [ ] TODOからナレッジへの変換機能
- TODOカードに「ナレッジ」ボタンを追加
- ボタン押下でTODOをナレッジテーブルに移動
